# Getting started with Polar and NestJS

## Clone the repository

```bash
npx degit polarsource/examples/with-nestjs ./with-nestjs
```

## How to use

1. Run the command below to copy the `.env.example` file:

```bash
cp .env.example .env
```

2. Run the command below to install project dependencies:

```bash
npm install
```

3. Run the NestJS application in development mode:

```bash
npm run start:dev
```

Or build and run in production mode:

```bash
npm run build
npm run start:prod
```
