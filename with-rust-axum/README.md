# Getting started with Polar and Rust Axum

## Clone the repository

```bash
npx degit polarsource/examples/with-rust-axum ./with-rust-axum
```

## How to use

1. Run the command below to copy the `.env.example` file:

```
cp .env.example .env
```

2. Run the command below to build the project:

```
cargo build
```

3. Run the Rust Axum application using the following command:

```
cargo run
```
