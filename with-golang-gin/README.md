# Getting started with Polar and Go (Gin)

## Clone the repository

```bash
npx degit polarsource/examples/with-golang-gin ./with-golang-gin
```

## How to use

1. Run the command below to copy the `.env.example` file:

```
cp .env.example .env
```

2. Run the command below to install project dependencies:

```
go mod download
```

3. Run the Go application using the following command:

```
go run main.go
```
